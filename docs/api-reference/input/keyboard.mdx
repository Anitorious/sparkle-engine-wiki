# Keyboard Input

Handle keyboard input through the `Input` class using key state queries.

---

- **Namespace:** Bliss.CSharp.Interact
- **Type:** `static class`

---

## Features

The keyboard input functionality provides:

- **Key state queries** — Check if keys are pressed, held, or released
- **Text input mode** — Capture typed text from the user
- **Frame-based input** — Query input state synchronized with the game loop

## Methods

### IsKeyPressed

```csharp
public static bool IsKeyPressed(KeyboardKey key, bool enableRepeat = false)
```

Checks if the specified keyboard key was pressed in the current frame.

**Parameters:**
- `key` — The [KeyboardKey](#keyboardkey) to check
- `enableRepeat` — If true, the key press will be considered a repeated action (default: `false`)

**Returns:** `true` if the key was pressed; otherwise, `false`

**Example:**

```csharp
if (Input.IsKeyPressed(KeyboardKey.Space)) {
    player.Jump();
}
```

---

### IsKeyDown

```csharp
public static bool IsKeyDown(KeyboardKey key)
```

Checks if the specified keyboard key is currently pressed down (held).

**Parameters:**
- `key` — The [KeyboardKey](#keyboardkey) to check

**Returns:** `true` if the key is held down; otherwise, `false`

**Example:**

```csharp
if (Input.IsKeyDown(KeyboardKey.W)) {
    player.MoveForward(deltaTime);
}
```

---

### IsKeyReleased

```csharp
public static bool IsKeyReleased(KeyboardKey key)
```

Checks if the specified keyboard key was released in the current frame.

**Parameters:**
- `key` — The [KeyboardKey](#keyboardkey) to check

**Returns:** `true` if the key was released; otherwise, `false`

**Example:**

```csharp
if (Input.IsKeyReleased(KeyboardKey.E)) {
    inventory.Close();
}
```

---

### IsKeyUp

```csharp
public static bool IsKeyUp(KeyboardKey key)
```

Determines whether the specified key is currently up (not pressed).

**Parameters:**
- `key` — The [KeyboardKey](#keyboardkey) to check

**Returns:** `true` if the key is up; otherwise, `false`

**Example:**

```csharp
if (Input.IsKeyUp(KeyboardKey.LControl)) {
    player.CanSprint = true;
}
```

```csharp
    player.MoveForward();
}

// Check for modifier key combination

    SaveGame();
}

// Check for function key
if (Input.IsKeyPressed(KeyboardKey.F1)) {
    ShowHelp();
}

// Check for numeric keypad
if (Input.IsKeyPressed(KeyboardKey.Keypad5)) {
    SelectItem();
}
```

## Text Input

### GetTypedText

```csharp
public static bool GetTypedText(out string text)
```

Retrieves any text that was typed since the last frame, but only while text input is active via `EnableTextInput()`.

**Parameters:**
- `text` — The typed text collected since the previous frame. Will be empty if no text was entered

**Returns:** `true` if any text was entered; otherwise, `false`

**Example:**

```csharp
if (Input.GetTypedText(out string typedText)) {
    searchBox.Text += typedText;
}
```

---

### IsTextInputActive

```csharp
public static bool IsTextInputActive()
```

Determines whether text input mode is currently active.

**Returns:** `true` if text input is active; otherwise, `false`

**Example:**

```csharp
if (!Input.IsTextInputActive()) {
    Input.EnableTextInput();
}
```

---

### EnableTextInput

```csharp
public static void EnableTextInput()
```

Activates text input mode, allowing the application to receive typed text events.

**Example:**

```csharp
public void OnUIFocused() {
    Input.EnableTextInput();
}
```

---

### DisableTextInput

```csharp
public static void DisableTextInput()
```

Disable the text input process, ending any active text input session.

**Example:**

```csharp
public void OnUIBlurred() {
    Input.DisableTextInput();
}
```
